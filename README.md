# 点菜APP

一个简单的点菜应用，分为店家和顾客界面。

## 功能特性

### 店家功能
- 添加菜品（支持图片上传）
- 修改菜品
- 删除菜品
- 查看订单

### 顾客功能
- 浏览菜单（按分类筛选）
- 加入购物车
- 提交订单
